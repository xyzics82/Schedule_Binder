# Life Binder Web

3P Binder형 자기경영 흐름을 참고해 만든 정적 웹앱 MVP입니다. 공식 3P바인더의 속지나 디자인을 복제하지 않고, 목표, 시간, 기록, 리뷰, 지식화를 웹 UI로 재해석했습니다.

## 실행 방법

`index.html` 파일을 브라우저로 열면 바로 사용할 수 있습니다. 별도 설치나 서버가 필요 없습니다.

## 포함 기능

- 오늘 실행판
- 주간 계획판
- 월간 로드맵
- 목표 관리
- 프로젝트 관리
- 할 일 관리
- 서브 바인더 노트
- 일간/주간 리뷰
- 시간 사용 통계
- 계획/실행 라인형 시간표
- 주간 화면 기본 진입
- 독자 설계 종이형 `페이퍼` 주간 시트
- 마우스 드래그 계획/실행 시간 라인 작성
- 실행 기록 형광 하이라이트
- 주간 체크박스 작성 및 제목 수정
- 취소/변경 계획 메모
- 오늘 탭의 compact 스케줄 요약
- 스케줄 항목별 연결 메모와 사진 첨부
- 주요 기능 예시 가이드
- JSON 내보내기

데이터는 브라우저 `localStorage`에 저장됩니다.
